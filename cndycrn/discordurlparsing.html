That instruction for how to view the source code of this page might differ across browsers, but the point is that this page doesn't actually have the same meta info that shows up in discord as an embed if you tack a period on the end of the link. :)

How To

To see what I'm talking about, post the following link in discord, including the period on the end.

Now post the link in discord without the period on the end.

Huh? Why does that happen?

Discord's code that parses the link and turns it into a hyperlink doesn't include the period, so it will always link to this page regardless of whether or not you include the period.

On the other hand, the code that parses links in your message and creates the embeds does include the period if it's there, which causes the discrepancy, as the the metadata in the embed is indeed from a different page

You can see this even more clearly by following the hyperlinked smiley face in the embed when you include the period, as that goes to the page that it actually used for the embed.
